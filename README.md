# stevetodman.com

Personal website and tools. Deployed via Cloudflare Pages.

- **Clinical Tools** - BP percentile calculator (AAP 2017 guidelines)
- **Clinic Resources** - Intake forms for HTN, dyslipidemia, syncope (password-protected)
